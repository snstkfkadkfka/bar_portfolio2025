UX/UI Design

01. BOSE 웹사이트 리뉴얼

Bose의 메인 페이지는 한 화면에 모든 내용을 담을 수 있도록 레이아웃을 최적화하였으며, 벤토 레이아웃을 사용하여 디자인하였습니다. 또한, 메인 화면에는 호버 시 애니메이션 효과를 추가해 사용자 경험을 향상시켰습니다.
사이트는 총 4개의 페이지로 구성되었습니다. 메인 페이지, 헤드셋 페이지, About 페이지, 로그인 페이지. 각 페이지는 모던하고 도시적인 느낌을 주며, 심플하고 자유로운 분위기를 강조했습니다. 디자인에는 퍼플, 레클리, 블랙, 화이트 컬러를 사용하여 세련된 느낌을 살렸습니다. 헤드셋 페이지는 버튼 클릭 시 해당 제품만 나타나도록 인터랙티브하게 제작되었습니다.

사용 프로그램

#피그마 #일러스트 #포토샵 #비주얼스튜디오코드(html, css, javascript, jQuery)

02. The Vintage Wine Merchants 웹사이트 리뉴얼

The Vintage Wine Merchants는 다양한 술을 고객에게 제공하는 회사로, 주로 와인을 판매합니다. 각 페이지는 모던하고 고급스러우며, 심플하면서도 전통적인 분위기를 표현하기 위해 레드, 버건디, 레드브라운 등의 컬러를 주로 사용하였습니다. 와인 페이지에서는 와인을 제작 국가별, 종류별, 가격별로 검색할 수 있는 UI 디자인이 적용되어 있습니다. 사이트는 메인 페이지, 와인 페이지, 상세 페이지, 검색 페이지의 4개의 페이지로 구성되어 있습니다

사용 프로그램

#피그마 #애프터이펙트 #프리미어 #비주얼스튜디오코드(html, css, javascript, jQuery)

03. Budsies 웹사이트 리뉴얼

Budsies는 아이들의 그림, 동화 일러스트, 사진 등을 기반으로 맞춤형 봉제인형을 제작해주는 회사입니다. 주로 아이들을 위한 선물로 구매되는 경우가 많아, 천진난만하고 아기자기한 느낌을 강조하기 위해 애니메이션과 일러스트를 디자인에 반영했습니다.
또한, 컬러풀하고 활기찬 분위기를 위해 파스텔 형광색인 오렌지, 페일 그린, 핑크를 사용하여 디자인을 제작했습니다.

사용 프로그램

#피그마 #일러스트 #포토샵 #비주얼스튜디오코드(html, css, javascript, jQuery)

04. Prelude Studio

이 사이트는 대전의 유명한 문구 브랜드 프렐류드 스튜디오를 바탕으로 제작되었습니다. 블로그 형식으로 심플하면서도 간단하게 디자인되었으며, 메인 페이지, About 페이지, 판매 페이지, 연락처 페이지 총 4개의 페이지로 구성되어 있습니다.
사이트는 실제 제품 사진과 매장 사진을 중심으로, 프렐류드 스튜디오의 브랜드 느낌을 잘 살렸습니다. 또한, 메인 컬러로 핑크색을 사용하여 눈에 띄면서도 여성스러운 분위기를 강조했습니다.

사용 프로그램

#포토샵 #일러스트 #비주얼스튜디오코드(html, scss)

05. 타Go 개인 프로젝트

타GO는 타슈 API, 네이버 Maps API, 식품 영양 성분 API를 활용한 어플리케이션으로, 타슈 자전거의 개수와 칼로리 소모를 계산해줍니다.
타슈 API를 이용해 자전거의 위치와 개수를 가져와 네이버 Maps에 마커로 표시했습니다. 또한 식품 API를 통해 음식명과 칼로리 데이터를 활용, 평균 자전거 운동에 따른 칼로리 소모량을 계산하고 순 칼로리 결과를 도출했습니다. 빌드에 백앤드가 필요한 타슈 api는 JSON 파일을 이용해 데이터를 처리했습니다.
앱은 메인 페이지와 칼로리 계산 페이지로 구성된 총 2페이지로 제작되었습니다. 디자인에서는 주황색을 주로 사용하여 활기차고 에너지 넘치는 느낌을 강조했습니다.

사용 프로그램

#피그마 #비주얼스튜디오코드(Vue3, scss)

06. 감정 일기장 클론 프로젝트

이 프로젝트는 React와 React Router DOM을 활용하여 제작된 감정일기 클론 프로젝트로, Home, Diary, New, Edit 페이지를 포함한 총 4개의 페이지로 구성되었습니다. 라우터, 네비게이트, 파라미터를 이용해 페이지 간 이동을 구현하고, 웹 스토리지를 사용해 일기 내용을 반영구적으로 저장할 수 있도록 최적화했습니다. 디자인은 스노우 컬러인 핑크와 블루를 사용하여 밝고 안정감 있는 느낌을 주었으며, 전반적인 속도 개선을 통해 사용자 경험을 향상시켰습니다.

사용 프로그램

#피그마 #비주얼스튜디오코드(React-vite, scss)
